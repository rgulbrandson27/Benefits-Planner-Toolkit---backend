# Benefits Planner Toolkit - Backend

RESTful API built with Spring Boot for SSDI work incentives planning and benefits calculations.

**Frontend Repository**: [Benefits Planner Toolkit - Frontend](link)

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Features
- Multi-tenant organization/client/scenario data model
- SSDI work incentives calculation endpoints
- Timeline data persistence (TWP, EPE, Medicare, SGA tracking)
- JSON serialization for complex calculation data structures

## API Design
Follows REST principles with DTO pattern for clean separation between API contracts and database entities.
